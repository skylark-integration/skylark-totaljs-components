## j-Range

- easy usage

__Attributes__:
- `data-max="100"` maximum value
- `data-min="0"` minimum value
- `data-step="1"` step size
- `data-required="true"` highlights the label as required
- `data-jc-type="number"` converts the value to number (otherwise will be `string`)
